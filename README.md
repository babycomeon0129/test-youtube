# event-vue_event_package

This template should help get you started developing with Vue 3 in Vite.

## 用途
test youtube list
DEMO頁，抓取youtube頻道、播放清單資訊

## API
/channels       頻道資訊
/playlistItems  播放清單
/videos         影片資訊
/commentThreads 留言
/playlists      所有播放清單，未實作，僅call api
